# Changelog

All notable changes to this project are documented here.
Format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

---

## [1.0.0] — 2024-11-15

### Added
- Full frequentist pipeline (z-test, t-test, Bonferroni correction)
- Bayesian Beta-Binomial model with Monte Carlo inference
- Pre-experiment power analysis and SRM check
- Synthetic data generator with realistic experiment simulation
- 5 publication-quality visualization functions
- Full unit test suite (94% coverage)
- Experiment design and statistical methods documentation

---

## [0.2.0] — 2024-10-30

### Added
- Bayesian analysis module
- Segment analysis (device, user type)
- Daily conversion rate time-series plot

### Fixed
- Novelty effect buffer now correctly excludes first 3 days

---

## [0.1.0] — 2024-10-15

### Added
- Initial project scaffold
- Frequentist z-test for conversion rates
- Basic data loader and synthetic data generation
